# MyRecipes App
MyRecipes is a clean and scalable Android recipe application built with Kotlin and Jetpack Compose, following Clean Architecture. It fetches recipes from the DummyJSON 
Recipes API using Ktor, utilizes Coroutines for asynchronous operations, and delivers a modern, responsive user experience.
